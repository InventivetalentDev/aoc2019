# Advent of Code 2019 solutions

## Installation

Using NPM:

```
npm i
```

Using Yarn:

```
yarn
```

## Running dev mode

```
npm start day<N>
```

or

```
yarn start day<N>
```

Example:

```
yarn start day1
```

If the day folder does not exist, it will be created from template.

## Running all solutions

```
npm start
```

or

```
yarn start
```

## Languages used:

- TypeScript
- JavaScript
